playhaven_hide_notification_badge

Hide a PlayHaven notification.

Syntax :

playhaven_hide_notification_badge(x, y, w, h, ident)


Returns : N/A


Description

With this function you can hide any PlayHaven notification badges that you have previously created with the function playhaven_add_notification_badge.


Example :

if global.game_state != st_mainmenu
   {
   playhaven_hide_notification_badge();
   }
else
   {
   playhaven_add_notification_badge(x- 16, y - 16, 32, 32, "games_widget");
   }

The above code will check the value of a global variable and either add or hide a PlayHaven notification badge.
