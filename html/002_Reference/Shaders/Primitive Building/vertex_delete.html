vertex_delete

Remove all vertex data from the given buffer.

Syntax :

vertex_delete(buffer);


ArgumentDescription
buffer The buffer to write the information to.


Returns: N/A


Description

This function will remove all vertex data from the given buffer. Note that if the buffer has been "frozen" using the vertex_freeze then you cannot use this function.


Example :

vertex_delete(buff);

The above code will remove all vertex data from the buffer indexed in the variable "buff".
