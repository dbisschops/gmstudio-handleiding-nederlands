vertex_rgba

Set the RGBA values for the vertex.

Syntax :

vertex_rgba(buffer, r, g, b, a);


ArgumentDescription
buffer The buffer to write the information to.
r The red component part of a colour (0 - 255).
g The green component part of a colour (0 - 255).
b The blue component part of a colour (0 - 255).
a The alpha value for the colour (0 - 255).


Returns: N/A


Description

This function will set the RGBA values for the vertex currently being defined for the custom primitive. You supply the buffer to write the data into as well as the red, green, blue and alpha values that you wish to use.


Example :

vertex_rgba(buff, 255, 128, 0, 0.5);

The above code will set the RGBA values of the vertex being defined.
